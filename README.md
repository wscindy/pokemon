# Pokemon TCG Online — Ruby on Rails Web App

A full-stack web application that digitalizes Pokemon Trading Card Game, implementing complete gameplay mechanics, real-time multiplayer battles, deck management, and matchmaking system with Vue.js frontend and Rails API backend.
